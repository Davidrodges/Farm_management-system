⚠️ PASSWORD CONFIGURATION INSTRUCTIONS ⚠️

You need to update the database password in ONE file on the server:

FILE TO EDIT: config/db.php
LINE NUMBER: 8

CURRENT LINE:
$password = "REPLACE_WITH_YOUR_INFINITYFREE_PASSWORD";

CHANGE IT TO:
$password = "your_actual_infinityfree_password_here";

---

HOW TO EDIT ON THE SERVER:
1. Go to InfinityFree File Manager
2. Navigate to: htdocs/config/
3. Right-click on: db.php
4. Click: Edit or Code Edit
5. Find line 8 (the password line)
6. Replace REPLACE_WITH_YOUR_INFINITYFREE_PASSWORD with your real password
7. Click Save

---

YOUR INFINITYFREE PASSWORD IS:
The password you use to log into infinityfree.net (your vPanel password)

---

AFTER EDITING:
Visit http://myfirstproject.rf.gd/db_test.php to verify the connection works.
